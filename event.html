LeQube Meetup Avril 2018

A eu lieu le 7 Avril 2018

Nous organisons en collaboration avec Solidar’IT Tech Hub organise notre tout premier meetup pour vous offrir une série de présentations sur 4 thèmes :

  • IoT & Big Data : Opportunités et Ouvertures
  • Dévélopment Web avec l’architecture MVC
  • Le Web de demain
  • IA : Communication homme-machine

Ce sera le 07 Avril 2018 à partir de 16h, venez découvrir, apprendre et partager avec des passionnés de tech comme vous.

Présentations

  • Dévélopment Web avec l’architecture MVC by Deschant Kounou (Slide)

  • Le Web de demain by Kenneth Assogba (Slide)

  • IA : Communication homme-machine by Seth-Pharès Gnavo (Slide)

  • IoT & Big Data : Opportunités et Ouvertures by Rachid Ali (Slide)
